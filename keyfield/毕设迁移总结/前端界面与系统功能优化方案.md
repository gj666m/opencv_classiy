# 前端界面与系统功能优化方案

> 创建时间：2026-04-19
> 目的：重构 FastAPI Web 界面，从单一上传识别升级为多页 Tab 系统，
>       增加模型对比、批量识别、OpenCV 管线可视化、识别历史等功能。
> 不涉及模型重新训练，只改前端界面 + 后端接口。

---

## 一、设计决策记录

| 决策项 | 选择 | 说明 |
|--------|------|------|
| 页面导航 | 顶部 Tab 标签页 | 单页应用，切换不刷新 |
| 模型对比展示 | 四宫格 + 表格 | 上方可视化，下方数据表 |
| OpenCV 演示 | 完整管线可视化 | 原图→去噪→CLAHE→最终输入，每步可见 |
| 图表库 | ECharts | CDN 引入，饼图/柱状图/雷达图 |
| 配色风格 | 清新环保风 | 绿色主色，白色背景，手绘风图标 |
| 历史记录 | 前端 localStorage | 关浏览器不丢失，不换电脑 |
| 首页内容 | 系统介绍卡片 + 快速上传入口 | 一进来就能用 |

### 4.3 接口详细设计

**`POST /api/compare`**

```python
# 伪代码
async def compare(file):
    image = read_image(file)
    results = []
    for model_key in AVAILABLE_MODELS:
        classifier = manager.get_classifier(model_key)
        t0 = time.time()
        result = classifier.predict_image(image)
        t1 = time.time()
        result["inference_time_ms"] = round((t1 - t0) * 1000, 1)
        result["model_key"] = model_key
        results.append(result)
    return {"results": results}
```

**`POST /api/batch-predict`**

```python
# 伪代码
async def batch_predict(files: List[UploadFile]):
    results = []
    for file in files:
        image = read_image(file)
        result = classifier.predict_image(image)
        result["filename"] = file.filename
        results.append(result)
    # 统计
    stats = compute_batch_stats(results)
    return {"results": results, "stats": stats}
```

**`POST /api/opencv-pipeline`**

```python
# 伪代码
async def opencv_pipeline(file):
    image = read_image(file)  # BGR
    steps = []
    # 步骤1: 原图
    steps.append({"name": "原始图片", "image": to_base64(image), "description": "OpenCV 读取的 BGR 原始图片"})
    # 步骤2: 高斯去噪
    denoised = denoise(image, method="gaussian", ksize=3)
    steps.append({"name": "高斯去噪", "image": to_base64(denoised), "params": "ksize=3, σ=0"})
    # 步骤3: CLAHE
    clahe_result = clahe(denoised, clip_limit=2.0)
    steps.append({"name": "CLAHE 增强", "image": to_base64(clahe_result), "params": "clipLimit=2.0, gridSize=8×8"})
    # 步骤4: 最终输入 (resize到224x224)
    resized = resize(clahe_result, (224, 224))
    steps.append({"name": "模型输入", "image": to_base64(resized), "description": "224×224 归一化后的模型输入"})
    return {"steps": steps}
```

**修改文件**：`ui/app.py`

2. 新增 `POST /api/batch-predict` — 批量识别，支持多文件上传，返回结果列表 + 统计摘要
